### Hardware requirements

The following requirements are recommended for running gitopia:

- 4 or more physical CPU cores
- At least 200GB of SSD disk storage
- At least 8GB of memory (RAM)
- At least 100mbps network bandwidth


### Install dependencies Required

```
sudo apt update && sudo apt upgrade -y && sudo apt install curl tar wget clang pkg-config libssl-dev jq build-essential bsdmainutils git make ncdu gcc git jq chrony liblz4-tool -y
```

### Install go

We are gonna use GO Version 1.20.4
If you already have GO installed you can skip this 

```
ver="1.20"
cd $HOME
wget "https://golang.org/dl/go$ver.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go$ver.linux-amd64.tar.gz"
rm "go$ver.linux-amd64.tar.gz"
echo "export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin" >> ~/.bash_profile
source ~/.bash_profile
go version
```

### Install binary

```
cd $HOME
rm -rf mun-node/
git clone https://github.com/munblockchain/mun-node
cd mun-node/
make build
```

Verify that you've installed mund successfully

```
mund version
```

### Prepare Cosmovisor
```
mkdir -p $HOME/.mun/cosmovisor/genesis/bin
mv build/mund $HOME/.mun/cosmovisor/genesis/bin/
rm -rf build


sudo ln -s $HOME/.mun/cosmovisor/genesis $HOME/.mun/cosmovisor/current
sudo ln -s $HOME/.mun/cosmovisor/current/bin/mund /usr/local/bin/mund
```

### Install Cosmovisor
```

go install cosmossdk.io/tools/cosmovisor/cmd/cosmovisor@v1.4.0
```

### Config app

```
mund config chain-id mun-1
mund config keyring-backend file
```

### Initialize Node

Please change `<YOUR_MONIKER>` To your own Moniker.

```
mund init <YOUR_MONIKER> --chain-id mun-1
```

### Download Genesis file and addrbook

- Genesis

```
curl -Ls https://snapshots.indonode.net/mun/genesis.json > $HOME/.mun/config/genesis.json
```

- Addrbook

```
curl -Ls https://snapshots.indonode.net/mun/addrbook.json > $HOME/.mun/config/addrbook.json
```

### Configure Seeds and Peers

```
SEEDS="036c564e3de76ffad3e013bea52c16eb1de5a400@31.14.40.112:26656"
PEERS=""
sed -i 's|^seeds *=.*|seeds = "'$SEEDS'"|; s|^persistent_peers *=.*|persistent_peers = "'$PEERS'"|' $HOME/.mun/config/config.toml
sed -i -e "s/^filter_peers *=.*/filter_peers = \"true\" /" $HOME/.mun/config/config.toml
```

### Configure Gas Prices
```
sed -i -e "s|^minimum-gas-prices *=.*|minimum-gas-prices = \"0umun\"|" $HOME/.mun/config/app.toml
```

### config pruning

```
PRUNING="custom"
PRUNING_KEEP_RECENT="100"
PRUNING_INTERVAL="19"

sed -i -e "s/^pruning *=.*/pruning = \"$PRUNING\"/" $HOME/.mun/config/app.toml
sed -i -e "s/^pruning-keep-recent *=.*/pruning-keep-recent = \
\"$PRUNING_KEEP_RECENT\"/" $HOME/.mun/config/app.toml
sed -i -e "s/^pruning-interval *=.*/pruning-interval = \
\"$PRUNING_INTERVAL\"/" $HOME/.mun/config/app.toml
```


### create service file and start node

Create service file
Create a `mund.service` file in the `/etc/systemd/system` folder with the following code. 
You can change `USER` with your Linux user name. You need sudo previlege to do this step.

```
sudo tee /etc/systemd/system/mund.service > /dev/null << EOF
[Unit]
Description=mun node service
After=network-online.target

[Service]
User=$USER
ExecStart=$(which cosmovisor) run start
Restart=on-failure
RestartSec=10
LimitNOFILE=65535
Environment="DAEMON_HOME=$HOME/.mun"
Environment="DAEMON_NAME=mund"
Environment="UNSAFE_SKIP_BACKUP=true"

[Install]
WantedBy=multi-user.target
EOF
sudo systemctl daemon-reload
sudo systemctl enable mund
```

### Download Latest Snapshot
```
curl -L https://snapshots.indonode.net/mun/mun-snapshot.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.mun
[[ -f $HOME/.mun/data/upgrade-info.json ]] && cp $HOME/.mun/data/upgrade-info.json $HOME/.mun/cosmovisor/genesis/upgrade-info.json
```

- Enable Service and Start Node

```
sudo systemctl restart mund
sudo journalctl -fu mund -o cat
```
