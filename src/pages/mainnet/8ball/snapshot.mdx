## Intruction

### Install lz4
```
sudo apt install lz4 -y
```

### Stop 8ball and Backup priv_validator_state.json
```
sudo systemctl stop 8ball
cp $HOME/.8ball/data/priv_validator_state.json $HOME/.8ball/priv_validator_state.json.backup
rm -rf $HOME/.8ball/data
```

### Download Snapshot
```
curl -L https://8ball.service.indonode.net/8ball-snapshot.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.8ball
mv $HOME/.8ball/priv_validator_state.json.backup $HOME/.8ball/data/priv_validator_state.json
```

### Restart Node and Check the logs

```
sudo systemctl restart 8ball && sudo journalctl -fu 8ball -o cat
```
