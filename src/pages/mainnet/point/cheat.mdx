### Planq CLI Cheatsheet

- Always be careful with the capitalized words
- Specify `--chain-id`

### Wallet Management

Add Wallet
Specify the value `<wallet>` with your own wallet name

```
pointd keys add <wallet>
```

Recover Wallet
```
pointd keys add <wallet> --recover
```

List Wallet
```
pointd keys list
```

Delete Wallet
```
pointd keys delete <wallet>
```

Check Wallet Balance
```
pointd q bank balances $(pointd keys show wallet -a)
```

### Validator Management

Please adjust `<wallet>` , `MONIKER` , `YOUR_KEYBASE_ID` , `YOUR_DETAILS` , `YOUUR_WEBSITE_URL`

Create Validator (Staking) with 1K Point
```
pointd tx staking create-validator \
--amount=100000000000000000000apoint \
--pubkey=$(pointd tendermint show-validator) \
--moniker=<MONIKER> \
--identity "YOUR_KEYBASE_ID" \
--website "YOUUR_WEBSITE_URL" \
--details "YOUR_DETAILS" \
--chain-id=point_10687-1 \
--commission-rate="0.05" \
--commission-max-rate="0.10" \
--commission-max-change-rate="0.01" \
--min-self-delegation="1" \
--gas="400000" \
--gas-prices="0.025apoint" \
--from=wallet
```

Edit Validator
```
pointd tx staking edit-validator \
--moniker="<MONIKER>" \
--identity="YOUR_KEYBASE_ID" \
--details="YOUR_DETAILS" \
--website="YOUR_WEBSITE_URL"
--chain-id point_10687-1 \
--commission-rate=0.05 \
--from=<wallet> \
--gas-adjustment="1.4" \
--gas="400000" \
-y
```


Unjail Validator
```
pointd tx slashing unjail --from <wallet> --chain-id --chain-id point_10687-1 --gas-adjustment 1.4 --gas="400000" --gas-prices="0.025apoint" -y
```

Check Jailed Reason
```
pointd query slashing signing-info $(pointd tendermint show-validator)
```

### Token Management

Withdraw Rewards
```
pointd tx distribution withdraw-all-rewards --from wallet --chain-id point_10687-1 --gas-adjustment 1.4 --gas="400000" --gas-prices="0.025apoint" -y
```

Withdraw Rewards with Comission
```
pointd tx distribution withdraw-rewards $(pointd keys show wallet --bech val -a) --commission --from wallet --chain-id point_10687-1 --gas-adjustment 1.4 --gas="400000" --gas-prices="0.025apoint" -y
```

Delegate Token to your own validator
```
pointd tx staking delegate $(pointd keys show wallet --bech val -a) 1000000apoint --from wallet --chain-id point_10687-1 --gas-adjustment 1.4 --gas="400000" --gas-prices="0.025apoint" -y
```

Delegate Token to other validator
```
pointd tx staking redelegate $(pointd keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000apoint --from wallet --chain-id point_10687-1 --gas-adjustment 1.4 --gas="400000" --gas-prices="0.025apoint" -y
```

Unbond Token from your validator
```
pointd tx staking unbond $(pointd keys show wallet --bech val -a) 1000000apoint --from wallet --chain-id point_10687-1 --gas-adjustment 1.4 --gas="400000" --gas-prices="0.025apoint" -y
```

Send Token to another wallet
```
pointd tx bank send wallet <TO_WALLET_ADDRESS> 1000000apoint --from wallet --chain-id point_10687-1
```

### Governance 

Vote
You can change the value of `yes` to `no`,`abstain`,`nowithveto`

```
pointd tx gov vote 1 yes --from wallet --chain-id point_10687-1 --gas-adjustment 1.4 --gas="400000" --gas-prices="0.025apoint" -y
```

### Other

Set Your own Custom Ports
You can change value `CUSTOM_PORT=21` To any other ports
```
CUSTOM_PORT=21
sed -i -e "s%^proxy_app = \"tcp://127.0.0.1:26658\"%proxy_app = \"tcp://127.0.0.1:${CUSTOM_PORT}658\"%; s%^laddr = \"tcp://127.0.0.1:26657\"%laddr = \"tcp://127.0.0.1:${CUSTOM_PORT}657\"%; s%^pprof_laddr = \"localhost:6060\"%pprof_laddr = \"localhost:${CUSTOM_PORT}060\"%; s%^laddr = \"tcp://0.0.0.0:26656\"%laddr = \"tcp://0.0.0.0:${CUSTOM_PORT}656\"%; s%^prometheus_listen_addr = \":26660\"%prometheus_listen_addr = \":${CUSTOM_PORT}660\"%" $HOME/.pointd/config/config.toml
sed -i -e "s%^address = \"tcp://0.0.0.0:1317\"%address = \"tcp://0.0.0.0:${CUSTOM_PORT}317\"%; s%^address = \":8080\"%address = \":${CUSTOM_PORT}080\"%; s%^address = \"0.0.0.0:9090\"%address = \"0.0.0.0:${CUSTOM_PORT}090\"%; s%^address = \"0.0.0.0:9091\"%address = \"0.0.0.0:${CUSTOM_PORT}091\"%" $HOME/.pointd/config/app.toml
```

Enable Indexing usually enabled by default
```
sed -i -e 's|^indexer *=.*|indexer = "kv"|' $HOME/.pointd/config/config.toml
```

Disable Indexing
```
sed -i -e 's|^indexer *=.*|indexer = "null"|' $HOME/.pointd/config/config.toml
```

Reset Chain Data
```
pointd tendermint unsafe-reset-all --home $HOME/.pointd --keep-addr-book
```

### Delete Node

WARNING! Use this command wisely 
Backup your key first it will remove Planq

```
sudo systemctl stop pointd && \
sudo systemctl disable pointd && \
rm /etc/systemd/system/pointd.service && \
sudo systemctl daemon-reload && \
cd $HOME && \
rm -rf .pointd && \
rm -rf $(which pointd)
```
