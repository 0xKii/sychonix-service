- Update every 6 Hour

## Intruction

### install lz4
```
sudo apt install lz4 snapd -y
```

### Instruction
```
sed -i.bak -E "s|^(enable[[:space:]]+=[[:space:]]+).*$|\1false|" ~/.lambdavm/config/config.toml

sudo systemctl stop lambdavm

rm -rf ~/.lambdavm/data

lambdavm tendermint unsafe-reset-all --home ~/.lambdavm/

SNAP_NAME=$(curl -s https://snapshots.nodestake.top/lambda/ | egrep -o ">20.*\.tar.lz4" | tr -d ">")

curl -o - -L https://snapshots.nodestake.top/lambda/${SNAP_NAME}  | lz4 -c -d - | tar -x -C $HOME/.lambdavm

sudo systemctl restart lambdavm

journalctl -f -u lambdavm
```
