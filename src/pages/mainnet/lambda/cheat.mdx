### Planq CLI Cheatsheet

- Always be careful with the capitalized words
- Specify `--chain-id`

### Wallet Management

Add Wallet
Specify the value `<wallet>` with your own wallet name

```
lambdavm keys add <wallet>
```

Recover Wallet
```
lambdavm keys add <wallet> --recover
```

List Wallet
```
lambdavm keys list
```

Delete Wallet
```
lambdavm keys delete <wallet>
```

Check Wallet Balance
```
lambdavm q bank balances $(lambdavm keys show wallet -a)
```

### Validator Management

Please adjust `<wallet>` , `MONIKER` , `YOUR_KEYBASE_ID` , `YOUR_DETAILS` , `YOUUR_WEBSITE_URL`

Create Validator (Staking)
```
lambdavm tx staking create-validator \
  --amount=20000000000000000000000ulamb \
  --pubkey=$(lambdavm tendermint show-validator) \
  --moniker="<Moniker>" \
  --chain-id=lambda_92000-1 \
  --commission-rate="0.05" \
  --commission-max-rate="0.20" \
  --commission-max-change-rate="0.01" \
  --min-self-delegation="1000000" \
  --gas="300000" \
  --gas-prices="0.025ulamb" \
  --from=<wallet> \
  --identity="" \
  --website="" \
  --details="" \
  --broadcast-mode block -y
```

Edit Validator
```
lambdavm tx staking edit-validator \
--moniker="<MONIKER>" \
--identity="YOUR_KEYBASE_ID" \
--details="YOUR_DETAILS" \
--website="YOUR_WEBSITE_URL"
--chain-id lambda_92000-1 \
--commission-rate=0.05 \
--from=<wallet> \
--gas-adjustment="1.15" \
--gas="300000" \
--gas-prices="0.025ulamb" \
-y
```


Unjail Validator
```
lambdavm tx slashing unjail --from <wallet> --chain-id --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

Check Jailed Reason
```
lambdavm query slashing signing-info $(lambdavm tendermint show-validator)
```

### Token Management

Withdraw Rewards
```
lambdavm tx distribution withdraw-all-rewards --from wallet --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

Withdraw Rewards with Comission
```
lambdavm tx distribution withdraw-rewards $(lambdavm keys show wallet --bech val -a) --commission --from wallet --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

Delegate Token to your own validator
```
lambdavm tx staking delegate $(lambdavm keys show wallet --bech val -a) 1000000ulamb --from wallet --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

Delegate Token to other validator
```
lambdavm tx staking redelegate $(lambdavm keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000ulamb --from wallet --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

Redelegate Token
```
lambdavm tx staking redelegate $(lambdavm keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000ulamb --from wallet --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

Unbond Token from your validator
```
lambdavm tx staking unbond $(lambdavm keys show wallet --bech val -a) 1000000ulamb --from wallet --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

Send Token to another wallet
```
lambdavm tx bank send wallet <TO_WALLET_ADDRESS> 1000000ulamb --from wallet --chain-id lambda_92000-1
```

### Governance 

Vote
You can change the value of `yes` to `no`,`abstain`,`nowithveto`

```
lambdavm tx gov vote 1 yes --from wallet --chain-id lambda_92000-1 --gas-adjustment 1.15 --gas="300000" --gas-prices="0.025ulamb" -y
```

### Other

Set Your own Custom Ports
You can change value `CUSTOM_PORT=15` To any other ports
```
CUSTOM_PORT=14
sed -i -e "s%^proxy_app = \"tcp://127.0.0.1:26658\"%proxy_app = \"tcp://127.0.0.1:${CUSTOM_PORT}658\"%; s%^laddr = \"tcp://127.0.0.1:26657\"%laddr = \"tcp://127.0.0.1:${CUSTOM_PORT}657\"%; s%^pprof_laddr = \"localhost:6060\"%pprof_laddr = \"localhost:${CUSTOM_PORT}060\"%; s%^laddr = \"tcp://0.0.0.0:26656\"%laddr = \"tcp://0.0.0.0:${CUSTOM_PORT}656\"%; s%^prometheus_listen_addr = \":26660\"%prometheus_listen_addr = \":${CUSTOM_PORT}660\"%" $HOME/.lambdavm/config/config.toml
sed -i -e "s%^address = \"tcp://0.0.0.0:1317\"%address = \"tcp://0.0.0.0:${CUSTOM_PORT}317\"%; s%^address = \":8080\"%address = \":${CUSTOM_PORT}080\"%; s%^address = \"0.0.0.0:9090\"%address = \"0.0.0.0:${CUSTOM_PORT}090\"%; s%^address = \"0.0.0.0:9091\"%address = \"0.0.0.0:${CUSTOM_PORT}091\"%" $HOME/.lambdavm/config/app.toml
```

Enable Indexing usually enabled by default
```
sed -i -e 's|^indexer *=.*|indexer = "kv"|' $HOME/.lambdavm/config/config.toml
```

Disable Indexing
```
sed -i -e 's|^indexer *=.*|indexer = "null"|' $HOME/.lambdavm/config/config.toml
```

Reset Chain Data
```
lambdavm tendermint unsafe-reset-all --home $HOME/.lambdavm --keep-addr-book
```

### Delete Node

WARNING! Use this command wisely 
Backup your key first it will remove Planq

```
sudo systemctl stop lambdavm && \
sudo systemctl disable lambdavm && \
rm /etc/systemd/system/lambdavm.service && \
sudo systemctl daemon-reload && \
cd $HOME && \
rm -rf .lambdavm && \
rm -rf $(which lambdavm)
```
