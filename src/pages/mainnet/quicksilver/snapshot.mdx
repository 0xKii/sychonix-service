## Intruction

### Install lz4
```
sudo apt install lz4 -y
```

### Stop Planqd and Backup priv_validator_state.json
```
sudo systemctl stop quicksilverd
cp $HOME/.quicksilverd/data/priv_validator_state.json $HOME/.quicksilverd/priv_validator_state.json.backup
rm -rf $HOME/.quicksilverd/data
```

### Download Snapshot
```
curl -L https://snapshots.indonode.net/quicksilver/quicksilver-snapshot.tar.lz4 | tar -Ilz4 -xf - -C $HOME/.quicksilverd
mv $HOME/.quicksilverd/priv_validator_state.json.backup $HOME/.quicksilverd/data/priv_validator_state.json
```

### Restart Node and Check the logs

```
sudo systemctl start quicksilverd && sudo journalctl -u quicksilverd -f --no-hostname -o cat
```
