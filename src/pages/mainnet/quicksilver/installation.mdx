### Hardware requirements

The following requirements are recommended for running gitopia:

- 4 or more physical CPU cores
- At least 200GB of SSD disk storage
- At least 8GB of memory (RAM)
- At least 100mbps network bandwidth


### Install dependencies Required

```
sudo apt update && sudo apt upgrade -y && sudo apt install curl tar wget clang pkg-config libssl-dev jq build-essential bsdmainutils git make ncdu gcc git jq chrony liblz4-tool -y
```

### Install go

We are gonna use GO Version 1.19.9
If you already have GO installed you can skip this 

```
ver="1.19.9"
cd $HOME
wget "https://golang.org/dl/go$ver.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go$ver.linux-amd64.tar.gz"
rm "go$ver.linux-amd64.tar.gz"
echo "export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin" >> ~/.bash_profile
source ~/.bash_profile
go version
```

### Install binary

```
cd $HOME
rm -rf quicksilver
git clone https://github.com/ingenuity-build/quicksilver.git
cd quicksilver
git checkout v1.2.12
make build
```

Verify that you've installed quicksilverd successfully

```
quicksilverd version
```

### Prepare Cosmovisor
```
mkdir -p $HOME/.quicksilverd/cosmovisor/genesis/bin
mv build/quicksilverd $HOME/.quicksilverd/cosmovisor/genesis/bin/
rm -rf build


sudo ln -s $HOME/.quicksilverd/cosmovisor/genesis $HOME/.quicksilverd/cosmovisor/current
sudo ln -s $HOME/.quicksilverd/cosmovisor/current/bin/quicksilverd /usr/local/bin/quicksilverd
```

### Install Cosmovisor
```

go install cosmossdk.io/tools/cosmovisor/cmd/cosmovisor@v1.4.0
```

### Config app

```
quicksilverd config chain-id quicksilver-2
quicksilverd config keyring-backend file
```

### Initialize Node

Please change `<YOUR_MONIKER>` To your own Moniker.

```
quicksilverd init <YOUR_MONIKER> --chain-id quicksilver-2
```

### Download Genesis file and addrbook

- Genesis

```
curl -Ls https://snapshots.indonode.net/quicksilver/genesis.json > $HOME/.quicksilverd/config/genesis.json
```

- Addrbook

```
curl -Ls https://snapshots.indonode.net/quicksilver/addrbook.json > $HOME/.quicksilverd/config/addrbook.json
```

### Configure Seeds and Peers

```
PEERS="$(curl -sS https://rpc.quicksilver.indonode.net/net_info | jq -r '.result.peers[] | "\(.node_info.id)@\(.remote_ip):\(.node_info.listen_addr)"' | awk -F ':' '{print $1":"$(NF)}' | sed -z 's|\n|,|g;s|.$||')"
sed -i -e "s|^persistent_peers *=.*|persistent_peers = \"$PEERS\"|" $HOME/.quicksilverd/config/config.toml
```

### Configure Gas Prices
```
sed -i -e "s|^minimum-gas-prices *=.*|minimum-gas-prices = \"0.0001uqck\"|" $HOME/.quicksilverd/config/app.toml
```

### config pruning

```
PRUNING="custom"
PRUNING_KEEP_RECENT="100"
PRUNING_INTERVAL="19"

sed -i -e "s/^pruning *=.*/pruning = \"$PRUNING\"/" $HOME/.quicksilverd/config/app.toml
sed -i -e "s/^pruning-keep-recent *=.*/pruning-keep-recent = \
\"$PRUNING_KEEP_RECENT\"/" $HOME/.quicksilverd/config/app.toml
sed -i -e "s/^pruning-interval *=.*/pruning-interval = \
\"$PRUNING_INTERVAL\"/" $HOME/.quicksilverd/config/app.toml
```


### create service file and start node

Create service file
Create a `quicksilverd.service` file in the `/etc/systemd/system` folder with the following code. 
You can change `USER` with your Linux user name. You need sudo previlege to do this step.

```
sudo tee /etc/systemd/system/quicksilverd.service > /dev/null << EOF
[Unit]
Description=quicksilver node service
After=network-online.target

[Service]
User=$USER
ExecStart=$(which cosmovisor) run start
Restart=on-failure
RestartSec=10
LimitNOFILE=65535
Environment="DAEMON_HOME=$HOME/.quicksilverd"
Environment="DAEMON_NAME=quicksilverd"
Environment="UNSAFE_SKIP_BACKUP=true"

[Install]
WantedBy=multi-user.target
EOF
sudo systemctl daemon-reload
sudo systemctl enable quicksilverd
```
