## Intruction

### Install lz4
```
sudo apt install lz4 -y
```

### Stop planqd and Backup priv_validator_state.json
```
sudo systemctl stop planqd
cp $HOME/.planqd/data/priv_validator_state.json $HOME/.planqd/priv_validator_state.json.backup
rm -rf $HOME/.planqd/data
```

### Download Snapshot
```
curl -L https://planq.service.indonode.net/planq-snapshot.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.planqd
mv $HOME/.planqd/priv_validator_state.json.backup $HOME/.planqd/data/priv_validator_state.json
```

### Restart Node and Check the logs

```
sudo systemctl restart planqd && sudo journalctl -fu planqd -o cat
```
