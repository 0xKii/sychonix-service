## Intruction

### install lz4
```
sudo apt install lz4 -y
```

### reset chain data
```
sudo systemctl stop marsd
marsd tendermint unsafe-reset-all --home $HOME/.mars --keep-addr-book
```

### download data

```
curl -o - -L https://snap.genznodes.dev/snapshot/mars/mars-1-data.tar.lz4 | lz4 -c -d - | tar -x -C $HOME/.mars
```

### start node

```
sudo systemctl restart marsd
sudo journalctl -fu marsd -o cat
```
