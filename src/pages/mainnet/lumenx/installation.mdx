### Hardware requirements

The following requirements are recommended for running Defund:

- 4 or more physical CPU cores
- At least 200GB of SSD disk storage
- At least 8GB of memory (RAM)
- At least 100mbps network bandwidth


### Install dependencies Required

```
sudo apt update && sudo apt upgrade -y && sudo apt install curl tar wget clang pkg-config libssl-dev jq build-essential bsdmainutils git make ncdu gcc git jq chrony liblz4-tool -y
```

### Install go

We are gonna use GO Version 1.19.5
If you already have GO installed you can skip this 

```
ver="1.19.5"
cd $HOME
wget "https://golang.org/dl/go$ver.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go$ver.linux-amd64.tar.gz"
rm "go$ver.linux-amd64.tar.gz"
echo "export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin" >> ~/.bash_profile
source ~/.bash_profile
go version
```

### Install binary

```
cd $HOME
git clone https://github.com/cryptonetD/lumenx.git
cd lumenx/
git checkout v1.4.0
make build
```

### Prepare Cosmovisor
```
mkdir -p $HOME/.lumenx/cosmovisor/genesis/bin
mv build/lumenxd $HOME/.lumenx/cosmovisor/genesis/bin/
rm -rf build
```
```
sudo ln -s $HOME/.lumenx/cosmovisor/genesis $HOME/.lumenx/cosmovisor/current
sudo ln -s $HOME/.lumenx/cosmovisor/current/bin/lumenxd /usr/local/bin/lumenxd
```

Verify that you've installed lumenxd successfully

```
lumenxd version
```

### Install Cosmovisor

```
go install cosmossdk.io/tools/cosmovisor/cmd/cosmovisor@v1.4.0
```

### Config app

```
lumenxd config chain-id LumenX
lumenxd config keyring-backend file
```

### Initialize Node

Please change `<YOUR_MONIKER>` To your own Moniker.

```
lumenxd init <YOUR_MONIKER> --chain-id LumenX
```

### Download Genesis file and addrbook

- Genesis

```
curl -Ls https://snapshots.indonode.net/lumenx/genesis.json > $HOME/.lumenx/config/genesis.json
```

- Addrbook

```
curl -Ls https://snapshots.indonode.net/lumenx/addrbook.json > $HOME/.lumenx/config/addrbook.json
```

### Configure Seeds and Peers

```
PEERS="$(curl -sS https://rpc.lumenx.indonode.net/net_info | jq -r '.result.peers[] | "\(.node_info.id)@\(.remote_ip):\(.node_info.listen_addr)"' | awk -F ':' '{print $1":"$(NF)}' | sed -z 's|\n|,|g;s|.$||')"
sed -i -e "s|^persistent_peers *=.*|persistent_peers = \"$PEERS\"|" $HOME/.lumenx/config/config.toml
```

### Set Minimum Gas Price
```
sed -i -e "s|^minimum-gas-prices *=.*|minimum-gas-prices = \"0.0025ulumen\"|" $HOME/.lumenx/config/app.toml
```

### Config pruning

```
PRUNING="custom"
PRUNING_KEEP_RECENT="100"
PRUNING_INTERVAL="19"

sed -i -e "s/^pruning *=.*/pruning = \"$PRUNING\"/" $HOME/.lumenx/config/app.toml
sed -i -e "s/^pruning-keep-recent *=.*/pruning-keep-recent = \
\"$PRUNING_KEEP_RECENT\"/" $HOME/.lumenx/config/app.toml
sed -i -e "s/^pruning-interval *=.*/pruning-interval = \
\"$PRUNING_INTERVAL\"/" $HOME/.lumenx/config/app.toml
```


### create service file and start node

Create service file
Create a `lumenxd.service` file in the `/etc/systemd/system` folder with the following code. 
You can change `USER` with your Linux user name. You need sudo previlege to do this step.

```
sudo tee /etc/systemd/system/lumenxd.service > /dev/null << EOF
[Unit]
Description=lumen node service
After=network-online.target

[Service]
User=$USER
ExecStart=$(which cosmovisor) run start
Restart=on-failure
RestartSec=10
LimitNOFILE=65535
Environment="DAEMON_HOME=$HOME/.lumenx"
Environment="DAEMON_NAME=lumenxd"
Environment="UNSAFE_SKIP_BACKUP=true"

[Install]
WantedBy=multi-user.target
EOF
sudo systemctl daemon-reload
sudo systemctl enable lumenxd
```

### Download Latest Snapshot
```
curl -L https://snapshots.indonode.net/lumenx/lumenx-snapshot.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.lumenx
[[ -f $HOME/.lumenx/data/upgrade-info.json ]] && cp $HOME/.lumenx/data/upgrade-info.json $HOME/.lumenx/cosmovisor/genesis/upgrade-info.json
```

- Enable Service and Start Node

```
sudo systemctl restart lumenxd
sudo journalctl -fu lumenxd -o cat
```
