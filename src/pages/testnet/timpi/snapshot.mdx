## Intruction

### Install lz4
```
sudo apt install lz4 -y
```

### Stop timpi and Backup priv_validator_state.json
```
sudo systemctl stop TimpiChain
cp $HOME/.TimpiChain/data/priv_validator_state.json $HOME/.TimpiChain/priv_validator_state.json.backup
rm -rf $HOME/.TimpiChain/data
```

### Download Snapshot
```
curl -L https://snapshots.indonode.net/timpi-t/timpi-snapshot.tar.lz4 | tar -Ilz4 -xf - -C $HOME/.TimpiChain
mv $HOME/.TimpiChain/priv_validator_state.json.backup $HOME/.TimpiChain/data/priv_validator_state.json
```

### Restart Node and Check the logs

```
sudo systemctl start TimpiChain && sudo journalctl -u TimpiChain -f --no-hostname -o cat
```
