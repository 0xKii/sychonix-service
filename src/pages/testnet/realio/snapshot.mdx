## Intruction
Updated every 12 at Midnight

### Install lz4
```
sudo apt install lz4 -y
```

### Stop Planqd and Backup priv_validator_state.json
```
sudo systemctl stop realio-networkd
cp $HOME/.realio-network/data/priv_validator_state.json $HOME/.realio-network/priv_validator_state.json.backup
rm -rf $HOME/.realio-network/data
```

### Download Snapshot
```
curl -L https://snapshots.indonode.net/realio-t/realio-t-snapshot.tar.lz4 | tar -Ilz4 -xf - -C $HOME/.realio-network
mv $HOME/.realio-network/priv_validator_state.json.backup $HOME/.realio-network/data/priv_validator_state.json
```

### Restart Node and Check the logs

```
sudo systemctl restart realio-networkd && sudo journalctl -u realio-networkd -f --no-hostname -o cat
```
