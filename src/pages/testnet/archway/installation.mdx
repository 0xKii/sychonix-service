### Hardware requirements

The following requirements are recommended

- 4 or more physical CPU cores
- At least 200GB of SSD disk storage
- At least 8GB of memory (RAM)
- At least 100mbps network bandwidth


### Install dependencies Required

```
sudo apt update && sudo apt upgrade -y && sudo apt install curl tar wget clang pkg-config libssl-dev jq build-essential bsdmainutils git make ncdu gcc git jq chrony liblz4-tool -y
```

### Install go

We are gonna use GO Version 1.19.9
If you already have GO installed you can skip this 

```
ver="1.19.9"
cd $HOME
wget "https://golang.org/dl/go$ver.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go$ver.linux-amd64.tar.gz"
rm "go$ver.linux-amd64.tar.gz"
echo "export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin" >> ~/.bash_profile
source ~/.bash_profile
go version
```

### Install binary

```
cd $HOME
rm -rf archway
git clone https://github.com/archway-network/archway.git
cd archway
git checkout v0.6.0
make build
```

### Prepare Cosmovisor
```
mkdir -p $HOME/.archway/cosmovisor/genesis/bin
mv build/archwayd $HOME/.archway/cosmovisor/genesis/bin/
rm -rf build


sudo ln -s $HOME/.archway/cosmovisor/genesis $HOME/.archway/cosmovisor/current
sudo ln -s $HOME/.archway/cosmovisor/current/bin/archwayd /usr/local/bin/archwayd
```

Verify that you've installed archwayd successfully

```
archwayd version
```

### Install Cosmovisor
```
go install cosmossdk.io/tools/cosmovisor/cmd/cosmovisor@v1.4.0
```

### Config app

```
archwayd config chain-id constantine-3
archwayd config keyring-backend test
```

### Initialize Node

Please change `<YOUR_MONIKER>` To your own Moniker.

```
archwayd init <YOUR_MONIKER> --chain-id constantine-3
```

### Download Genesis file and addrbook

- Genesis

```
curl -Ls https://snapshots.indonode.net/archway-t/genesis.json > $HOME/.archway/config/genesis.json
```

- Addrbook

```
curl -Ls https://snapshots.indonode.net/archway-t/addrbook.json > $HOME/.archway/config/addrbook.json
```

### Configure Seeds and Peers

```
PEERS="$(curl -sS https://rpc.archway-t.indonode.net/net_info | jq -r '.result.peers[] | "\(.node_info.id)@\(.remote_ip):\(.node_info.listen_addr)"' | awk -F ':' '{print $1":"$(NF)}' | sed -z 's|\n|,|g;s|.$||')"
sed -i -e "s|^persistent_peers *=.*|persistent_peers = \"$PEERS\"|" $HOME/.archway/config/config.toml
```

### Set Minimum Gas Prices

```
sed -i -e "s|^minimum-gas-prices *=.*|minimum-gas-prices = \"0aconst\"|" $HOME/.archway/config/app.toml
```

### config pruning

```
PRUNING="custom"
PRUNING_KEEP_RECENT="100"
PRUNING_INTERVAL="19"

sed -i -e "s/^pruning *=.*/pruning = \"$PRUNING\"/" $HOME/.archway/config/app.toml
sed -i -e "s/^pruning-keep-recent *=.*/pruning-keep-recent = \
\"$PRUNING_KEEP_RECENT\"/" $HOME/.archway/config/app.toml
sed -i -e "s/^pruning-interval *=.*/pruning-interval = \
\"$PRUNING_INTERVAL\"/" $HOME/.archway/config/app.toml
```


### create service file and start node

Create service file
Create a `archwayd.service` file in the `/etc/systemd/system` folder with the following code. 
You can change `USER` with your Linux user name. You need sudo previlege to do this step.

```
sudo tee /etc/systemd/system/archwayd.service > /dev/null << EOF
[Unit]
Description=archway-tesnet node service
After=network-online.target

[Service]
User=$USER
ExecStart=$(which cosmovisor) run start
Restart=on-failure
RestartSec=10
LimitNOFILE=65535
Environment="DAEMON_HOME=$HOME/.archway"
Environment="DAEMON_NAME=archwayd"
Environment="UNSAFE_SKIP_BACKUP=true"

[Install]
WantedBy=multi-user.target
EOF
sudo systemctl daemon-reload
sudo systemctl enable archwayd
```

### Download Latest Snapshot
```
curl -L https://snapshots.indonode.net/archway-t/archway-snapshot.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.archway
[[ -f $HOME/.archway/data/upgrade-info.json ]] && cp $HOME/.archway/data/upgrade-info.json $HOME/.archway/cosmovisor/genesis/upgrade-info.json
```

- Enable Service and Start Node

```
sudo systemctl restart archwayd
sudo journalctl -fu archwayd -o cat
```
