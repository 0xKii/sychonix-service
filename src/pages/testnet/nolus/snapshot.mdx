## Intruction


### Install lz4
```
sudo apt install lz4 -y
```

### Stop Nolusd and Backup priv_validator_state.json
```
sudo systemctl stop nolusd
cp $HOME/.nolus/data/priv_validator_state.json $HOME/.nolus/priv_validator_state.json.backup
rm -rf $HOME/.nolus/data
```

### Download Snapshot
```
curl -L https://nolus-t.service.indonode.net/nolus-snapshot.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.nolus
mv $HOME/.nolus/priv_validator_state.json.backup $HOME/.nolus/data/priv_validator_state.json
```

### Restart Node and Check the logs

```
sudo systemctl restart nolusd && sudo journalctl -u nolusd -f --no-hostname -o cat
```
