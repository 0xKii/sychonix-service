### CLI Cheatsheet

- Always be careful with the capitalized words
- Specify `--chain-id`

### Wallet Management

Add Wallet
Specify the value `<wallet>` with your own wallet name

```
cored keys add <wallet>
```

Recover Wallet
```
cored keys add <wallet> --recover
```

List Wallet
```
cored keys list
```

Delete Wallet
```
cored keys delete <wallet>
```

Check Wallet Balance
```
cored q bank balances $(cored keys show wallet -a)
```

### Validator Management

Please adjust `<wallet>` , `MONIKER` , `YOUR_KEYBASE_ID` , `YOUR_DETAILS` , `YOUUR_WEBSITE_URL`

Create Validator (Staking)
```
cored tx staking create-validator \
--amount=30000000000utestcore \
--pubkey=$(cored tendermint show-validator) \
--moniker="YOUR_MONIKER_NAME" \
--identity="YOUR_KEYBASE_ID" \
--details="YOUR_DETAILS" \
--website="YOUR_WEBSITE_URL" \
--chain-id=coreum-testnet-1 \
--commission-rate=0.05 \
--commission-max-rate=0.20 \
--commission-max-change-rate=0.01 \
--min-self-delegation=20000000000 \
--from=wallet \
--gas auto \
--fees 5000utestcore \
-y
```

Edit Validator
```
cored tx staking edit-validator \
--moniker="<MONIKER>" \
--identity="YOUR_KEYBASE_ID" \
--details="YOUR_DETAILS" \
--website="YOUR_WEBSITE_URL"
--chain-id coreum-testnet-1 \
--commission-rate=0.05 \
--from=<wallet> \
--gas auto \
--fees 5000utestcore \
-y
```


Unjail Validator
```
cored tx slashing unjail --from wallet --chain-id --chain-id coreum-testnet-1 --gas auto --fees 5000utestcore -y
```

Check Jailed Reason
```
cored query slashing signing-info $(cored tendermint show-validator)
```

### Token Management

Withdraw Rewards
```
cored tx distribution withdraw-all-rewards --from wallet --chain-id coreum-testnet-1 --gas auto --fees 5000utestcore -y
```

Withdraw Rewards with Comission
```
cored tx distribution withdraw-rewards $(cored keys show wallet --bech val -a) --commission --from wallet --chain-id coreum-testnet-1 --gas auto --fees 5000utestcore -y
```

Delegate Token to your own validator
```
cored tx staking delegate $(cored keys show wallet --bech val -a) <AMMOUNT>utestcore --from wallet --chain-id coreum-testnet-1  --gas auto --fees 5000utestcore -y
```

Delegate Token to other validator
```
cored tx staking redelegate $(cored keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> <AMMOUNT>utestcore --from wallet --chain-id coreum-testnet-1 --gas auto --fees 5000utestcore -y
```

Unbond Token from your validator
```
cored tx staking unbond $(cored keys show wallet --bech val -a) <AMMOUNT>utestcore --from wallet --chain-id coreum-testnet-1  --gas auto --fees 5000utestcore -y
```

Send Token to another wallet
```
cored tx bank send wallet <TO_WALLET_ADDRESS> <AMMOUNT>utestcore --from wallet --chain-id coreum-testnet-1
```

### Governance 

Vote
You can change the value of `yes` to `no`,`abstain`,`nowithveto`

```
cored tx gov vote 1 yes --from wallet --chain-id coreum-testnet-1  --gas auto --fees 5000utestcore -y
```

### Other

Set Your own Custom Ports
You can change value `CUSTOM_PORT=31` To any other ports
```
CUSTOM_PORT=31
sed -i.bak -e "s%^proxy_app = \"tcp://127.0.0.1:26658\"%proxy_app = \"tcp://127.0.0.1:${CUSTOM_PORT}658\"%; s%^laddr = \"tcp://127.0.0.1:26657\"%laddr = \"tcp://127.0.0.1:${CUSTOM_PORT}657\"%; s%^pprof_laddr = \"localhost:6060\"%pprof_laddr = \"localhost:${CUSTOM_PORT}060\"%; s%^laddr = \"tcp://0.0.0.0:26656\"%laddr = \"tcp://0.0.0.0:${CUSTOM_PORT}656\"%; s%^prometheus_listen_addr = \":26660\"%prometheus_listen_addr = \":${CUSTOM_PORT}660\"%" $HOME/.core/coreum-testnet-1/config/config.toml
sed -i.bak -e "s%^address = \"tcp://0.0.0.0:1317\"%address = \"tcp://0.0.0.0:${CUSTOM_PORT}317\"%; s%^address = \":8080\"%address = \":${CUSTOM_PORT}080\"%; s%^address = \"0.0.0.0:9090\"%address = \"0.0.0.0:${CUSTOM_PORT}090\"%; s%^address = \"0.0.0.0:9091\"%address = \"0.0.0.0:${CUSTOM_PORT}091\"%; s%^address = \"0.0.0.0:8545\"%address = \"0.0.0.0:${CUSTOM_PORT}545\"%; s%^ws-address = \"0.0.0.0:8546\"%ws-address = \"0.0.0.0:${CUSTOM_PORT}546\"%" $HOME/.core/coreum-testnet-1/config/app.toml
```

Enable Indexing usually enabled by default
```
sed -i -e 's|^indexer *=.*|indexer = "kv"|' $HOME/.core/coreum-testnet-1/config/config.toml
```

Disable Indexing
```
sed -i -e 's|^indexer *=.*|indexer = "null"|' $HOME/.core/coreum-testnet-1/config/config.toml
```

Reset Chain Data
```
cored tendermint unsafe-reset-all --home $HOME/.core/coreum-testnet-1 --keep-addr-book
```

### Delete Node

WARNING! Use this command wisely 
Backup your key first it will remove Cored from your system

```
sudo systemctl stop cored && \
sudo systemctl disable cored && \
rm /etc/systemd/system/cored.service && \
sudo systemctl daemon-reload && \
cd $HOME && \
rm -rf .core && \
rm -rf $(which cored)
```
