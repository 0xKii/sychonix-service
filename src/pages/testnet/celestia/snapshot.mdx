## Intruction


### Install lz4
```
sudo apt install lz4 -y
```

### Stop celestia-appd and Backup priv_validator_state.json
```
sudo systemctl stop celestia-appd
cp $HOME/.celestia-appd/data/priv_validator_state.json $HOME/.celestia-appd/priv_validator_state.json.backup
rm -rf $HOME/.celestia-appd/data
```

### Download Snapshot
```
curl -L https://snapshots.kjnodes.com/celestia-testnet/snapshot_latest.tar.lz4 | tar -Ilz4 -xf - -C $HOME/.celestia-appd
mv $HOME/.celestia-appd/priv_validator_state.json.backup $HOME/.celestia-appd/data/priv_validator_state.json
```

### Restart Node and Check the logs

```
sudo systemctl restart celestia-appd && sudo journalctl -u celestia-appd -f --no-hostname -o cat
```
